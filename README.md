# Mini-jvm

# PRE
- maven 3.3+
- jdk 1.8+
- graalvm 1.0+

# Build
```bash
mvn clean package

./build.sh # build binary executable file named mini-jvm.

```

# Install
```bash
mv mini-jvm ~/.local/bin
```
