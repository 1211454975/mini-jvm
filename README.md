使用 Java 8 实现 JVM

## 动机
1. 尝试了解 JVM 原理
2. 纸上得来终觉浅, 实践
3. 用简单的代码帮助 javaer 理解 JVM

## 规划
- Class 文件解析 80%
- 字节码执行 60%
- 类加载 50%
- 方法调用 30%
- 实例化 20%
- native 方法 0%
- 异常处理 0%
- 多线程 0%
- self-booting 0%

## 局限
1. 不实现 GC
2. 不实现 多线程
3. 不实现反射
4. 个人 Demo
